# 文件用途清单

本清单用于快速理解仓库内各文件职责。说明：未列出
`.git/`、`.next/`、`node_modules/` 等生成或系统目录。

## 根目录

- `.gitignore` - Git 忽略规则清单
- `.node-version` - Node 版本锁定（供 nodenv/asdf 等工具）
- `.npmrc` - npm/pnpm 项目级配置
- `.nvmrc` - nvm 使用的 Node 版本
- `.editorconfig` - 编辑器通用格式约定
- `.prettierignore` - Prettier 忽略路径
- `.stylelintignore` - Stylelint 忽略路径
- `.vscode/settings.json` - VS Code 团队统一编辑器设置
- `commitlint.config.cjs` - Commitlint 规则配置
- `components.json` - shadcn/ui 组件生成与别名配置
- `cz-git.config.js` - cz-git/commitizen 交互式提交配置
- `eslint.config.mjs` - ESLint 配置
- `lint-staged.config.cjs` - lint-staged 提交前检查配置
- `next-env.d.ts` - Next.js 自动生成的类型声明
- `next.config.ts` - Next.js 构建与运行配置
- `package.json` - 项目元信息、脚本与依赖
- `pnpm-lock.yaml` - pnpm 依赖锁文件
- `pnpm-workspace.yaml` - pnpm workspace 配置
- `postcss.config.mjs` - PostCSS/Tailwind 构建配置
- `prettier.config.cjs` - Prettier 配置
- `README.md` - 项目说明与基础用法
- `stylelint.config.cjs` - Stylelint 规则配置
- `tsconfig.json` - TypeScript 编译配置
- `tsconfig.tsbuildinfo` - TypeScript 增量编译缓存（本地生成）
