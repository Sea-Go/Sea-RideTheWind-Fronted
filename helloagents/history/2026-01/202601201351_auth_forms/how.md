# 技术设计: 登录与注册表单组件

## 技术方案

### 核心技术

- Next.js (App Router) + React
- TypeScript
- Tailwind CSS + shadcn/ui

### 实现要点

- 使用受控输入管理用户名/密码/确认密码
- 本地校验逻辑与错误提示（模拟 API 结果）
- 密码可见性切换按钮，保持与 Vue 版本交互一致
- 提交成功后通过 `useRouter` 跳转 `/explore`
- 预留 API 调用位置，暂不接入状态管理

## 安全与性能

- **安全:** 基础输入校验，错误提示避免泄露账号存在性
- **性能:** 表单为轻量交互，无额外性能风险

## 测试与部署

- **测试:** 手动验证表单校验、可见切换与跳转
- **部署:** 无额外部署流程
